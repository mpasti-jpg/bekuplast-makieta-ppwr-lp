# bekuplast — makieta Landing Page /ppwr/

Makieta (wireframe HTML) strony docelowej kampanii **PPWR** dla bekuplast.pl.

- **Plik:** [`makieta_LP_ppwr.html`](makieta_LP_ppwr.html) — pojedynczy, samodzielny plik (HTML + CSS + odrobina JS, bez build-stepu)
- **Design system:** zielony bekuplast (`#65b32e`), Manrope / Inter / JetBrains Mono
- **Spójność:** wizualnie nawiązuje do makiet stron branżowych

## Podgląd

Otwórz `makieta_LP_ppwr.html` w przeglądarce, albo skorzystaj z GitHub Pages (jeśli włączone).

## Zakres makiety

Hub kampanii PPWR: hero + countdown, sekcje informacyjne (PPWR w skrócie, 5 punktów, oś czasu),
karty branż, zaproszenia do dwóch narzędzi (kalkulator kosztu i audyt gotowości), FAQ, dowody,
produkty, formularz „rozmowa z ekspertem", powiązane treści. Boczna nawigacja po sekcjach
i pasek CTA przyklejony do dołu okna.

> Status: makieta do akceptacji. Teksty są propozycją; fakty regulacyjne oparte na rozporządzeniu (UE) 2025/40.
